Why do all commands start with trafikito_?
For higher security standards. By using only alias commands on your server Trafikito ensures that no unexpected commands run. Imagine losing your login details or having an angry person on your team with an admin role.

When to use string display types?
When you want to have unmodified values on your dashboard. Display type text doesnâ€™t have options to modify values in any way. It will just use raw values from the command output.

When to use percent display types?
When you have a numeric value and want to get percentage representation of it. Just like all numeric types, you can set error and warning ranges for this type and use those in the notification settings.

When to use number display types?
When you have a numeric value and you want to set how many decimal places it should display on the dashboard and also you want to define warning and error ranges for the value.

When to use base display types?
When you have numeric values to track and it needs to be divided by some base. You can also add some suffix next to the value. A good example is the size of bytes. If the command return bytes and you want to see Megabytes or Gigabytes, you have to use display type: base.
