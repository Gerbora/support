  2. Select Dashboard
  3. In the servers list click Open next to the server name
  4. In the single server view click Settings on the top right
  5. If data sources are collapsed, click View next to it
  6. Click on the data source you want to edit
  7. See the list of variables and click Add Variable to add new

How to add a new data source?
  1. Open the main menu on the top left [menu icon]
  2. Select Dashboard
  3. In the servers list find the server you want to edit and click Open next to it
  4. On the top right click Settings
  5. On the bottom right click the round yellow button with the plus sign.
  6. Make sure Data source is selected
  7. Enter a data source name and select command
  8. Click Add new data source to save

How to add a new command?
  1. Login to your machine. You can use SSH, FTP or anything else depending on your machine.
  2. Edit this file: /opt/trafikito/available_commands.sh
    1. If this file is not available, Trafikito agent is not installed yet or it is installed at a custom location
  3. Add a new alias of command at the end of the file.
  4. Save the file and wait a couple minutes.
