--- layout: post title: "How to switch server on the single server view without going back to the servers list?" permalink: /how-to-switch-server-on-the-single-server-view-without-going-back-to-the-servers-list.html tags: dashboard ---

While in the single server view with boxes of text and charts, scroll all the way up to where there is a button with the current selected server name on it.

Click on this name to view dropdown of all the servers in a workspace. Select any to switch.
