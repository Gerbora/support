--- layout: post title: "What are possible range changes to detect?" permalink: /what-are-possible-range-changes-to-detect.html tags: notifications ---

You can detect any possible value of a selected variable range change. You can select these ranges in the when case options while editing notifications.

  • Normal → warn
  • Normal → error
  • Normal → any
  • Warn → normal
  • Warn → error
  • Warn → any
  • Error → normal
  • Error → warn
  • Error → any
  • Any → normal
  • Any → warn
  • Any → error
  • Any → any
