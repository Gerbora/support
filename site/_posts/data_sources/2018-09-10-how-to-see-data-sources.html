--- layout: post title: "How to see data sources?" permalink: /how-to-see-data-sources.html tags: data sources ---

All data sources are visible in the settings of the server.

  1. Click the menu icon on the top left
  2. Go to Dashboard
  3. Click Open next to the server name you want to edit
  4. Click Settings on the top right
  5. Click Show next to Data sources

Here you can see the list of all data sources and variables defined inside them. Click on any item to open full details. You can change the data source name, command, data splitting to table options, variables, and variable display settings.
